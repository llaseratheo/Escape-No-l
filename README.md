
Escape Noël : Les Copains de Levier à l'Aventure

MODE D'EMPLOI RAPIDE
1) Remplacez les fichiers placeholder dans assets/audio/ et assets/images/ par les fichiers réels.
2) Déployez sur Vercel (ou Netlify). index.html doit être à la racine.
3) Utilisez 'Thégoat' comme mot de passe admin dans toute UI que vous ajouterez.

Contenu :
- index.html (intro + bouton)
- conclusion.html
- css/, js/
- assets/images (intro.png, background.png, outro.png, enigme1..7)
- assets/audio (intro.mp3, enigme1..7.mp3, final.mp3, intro.wav, background.wav, final.wav, click.wav, success.wav, error.wav)
